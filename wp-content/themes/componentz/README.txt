== Description ==
-----------------

Inspired by material design, based on flexible frameworks, designed with a passion for details. The theme you were looking for if you are an advanced developer or just a web enthusiast.

== Notes ==
---------------

- The menus contain up to 4 level hierarchy
- Header text feature is disabled
- The website tagline can be shown after the Kirki Customizer Framework plugin is installed. Option is then available in Customize -> Blog -> Blog Header Content - Jumbotron -> Website Tagline
